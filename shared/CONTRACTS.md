# Shared constants and types between frontend and backend

## Intervention Types
- `socratica` — Socratic Voice
- `paradosso` — Paradox
- `lente_filosofica` — Philosophical Lens

## User Roles
- `owner` — Full control
- `admin` — Manage users, config workspace
- `editor` — Write, edit, use AI features
- `reviewer` — Read, comment, use Lenses (read-only)

## User Reaction Types
- `accepted` — User modified text after intervention
- `ignored` — User didn't react
- `modified` — User partially accepted
- `rejected` — User explicitly rejected
